## datasets(DAGM2007*)
The link to the reconstructed DAGM2007 dataset is as follows: [DAGM2007*](https://www.kaggle.com/datasets/amor000/reconstructed-dagm2007-dataset).

```
python tools/train.py config/tinyvit.py
```
